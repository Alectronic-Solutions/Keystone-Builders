# Contractor Website Patterns

## Trust Signals (required on every contractor site)
- License number visible in header or trust bar
- Insurance badge (General Liability + Workers Comp mentioned)
- BBB/Angi/HomeAdvisor logo placeholders
- "Serving [City] since [Year]" in footer
- Phone number in navbar (click-to-call on mobile)

## Estimate Form Fields (standard)
- Name, Phone, Email
- Project type (dropdown: New Home / Remodel / Addition / Commercial / Other)
- Project description (textarea)
- Timeline (dropdown: ASAP / 1-3 months / 3-6 months / Planning ahead)
- Budget range (optional dropdown)

## Project Case Study Template
Each /projects/[slug] page needs:
- Hero image (full bleed)
- Project stats bar: Location | Square Footage | Timeline | Value
- Before/after or progress images (3-4)
- Challenge → Solution → Result narrative
- CTA at bottom: "Start Your Project"

## Mobile Requirements (contractor-specific)
- Phone number must be tap-to-call (tel: link) in sticky header
- Estimate form must work one-handed on mobile
- Project images must not be tiny on mobile (min 300px height)
- CTA button minimum 48px touch target
